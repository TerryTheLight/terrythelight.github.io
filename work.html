Work

What I've done.

Here's some selections of work that I've done. It comes in a variation forms of games and videos.

Need To Slow

Racing

The objective of this game is to finish the race last instead of first. Player will need to collect items that will slow down their car speed which is not controlable by the player.

Defense of The Tower

Strategy

An individual project & my first time of using the game engine Unity, player is required to destroy the enemies by typing the correct alphabet that appear on top of the enemy while avoiding them.

Jetride Joypack

Endless runner

In a team of 4 and using Unity Engine, this is a endless runner game where player will control the player using a joystick controller and avoiding all of the obstacles/traps while collecting fuels to fly even further.

Clash Nations

Strategy

A group project, player uses the mouse cursor to collect gold and deploy troops to defend and attack the enemy tower.

Blackout

Strategy Platfomer

A game prototype using Processing 3 to explain the gameplay of our game design. Basically the game is simple which is to collect items which will prevent the screen from getting covered by the black screen which will fall down as time passes by.

Roda Fantasi

Strategy Word Game

Game that I've created in a team of 3 programmers during my internship. It is up on the android and iOS app store. This game requires the player to guess the correct word that is generated from the game database. Player can either spin the wheel and earn game currencies if they guessed the correct letter in the word or buy vowels to get more insight on is the word generated.

Download on Google Store

Kluno: A Kid's Dream

Action Puzzle Game

A fast paced match three game. This first intellectual property by Gameka combines both tower defense and match three mechanic to create a brand new fast paced game. Focuses on the backend side, responsible for reading and writing data to the server, getting leaderboard and also login using google play or game services.

Download on Google Store

Kluno: Oogly's Escape

Action Puzzle Game

Inspired by Kluno: A Kid's Dream come Kluno Oogly's Escape with a whole new different gameplay and mechanic. My role is to help out with the backend and also the front end of the game. Learned a lot about Unity UI system here which includes Unity Grid layout, anchoring, and also scroll rect during the production of this game. Uses a lot of Unity animation system and also animator.

Download on Google Store

Kluno: Hero Battle

Action Puzzle Game

Not a kid nor an escape, is a HERO BATTLE! Kluno: Hero Battle is a realtime multiplayer game that is fast paced thus required a very high concentration and agility in order to excel. Focuses on the backend and the front end part of the game. During production period, practices and learned a lot of Object-oriented programming and how to make dynamic script in this project. Also learned on how to set-up and publish a game to google play store and apple store. Did some of the mobile optimization such as script caching.

Download on Google Store
